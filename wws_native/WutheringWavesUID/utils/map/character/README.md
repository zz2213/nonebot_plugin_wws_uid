# calc模版选择器v1.0

## 类型

``` 
ph -> 声骸套装（默认5件套)
生命 -> 生命值
攻击 -> 攻击力
防御 -> 防御力
共鸣效率 -> 共鸣效率
治疗效果加成 -> 治疗
```

## 操作符

```
普通操作符
= -> 等于
< -> 小于
> -> 大于
>= -> 大于等于
<= -> 小于等于
!= -> 不等于
in -> 包含
!in -> 不包含

逻辑操作符
&& -> 并且
|| -> 或者
! -> 非
```
